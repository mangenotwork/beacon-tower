#### 基于UDP的网络传输协议

##### 特点: 无连接，延迟小、数据传输效率高，支持广播多点传输,实时性高；

##### 使用注意: 传输小数据包应用场景，

##### 设计:

beacon-tower udp 是如何提升安全性?
1. 采用连接机制
2. IP黑白名单
3. 采用签名权限认证
4. 数据加解密


beacon-tower udp 是如何提升可靠性?
- 时间轮机制（我们多久未联系过？超过这个时间轮就需要确认对方是否存在）
- 数据传输确认交互 （最大能力保障传输数据完整性）
- 数据积压机制
- 数据重传机制


beacon-tower udp 其他特性?
- 限流
- 数据压缩    接入 gzip  （数据限定大小， 大于则进行拆分）
